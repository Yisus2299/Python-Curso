# Project 1 — Band Name Generator

## Overview

A beginner console program that generates a creative band name by combining two pieces of user input.

## What Was Built

An interactive script that asks the user for their childhood city and the name of a favorite pet or animal, then prints a suggested band name using both answers.

## Concepts Covered

- `input()` for user interaction
- Variables and string concatenation
- f-strings for formatted output

## Files

| File | Description |
|------|-------------|
| `project_1.py` | Main entry point |

## How to Run

```bash
python project_1.py
```

## How It Works

1. The program prompts for a city name.
2. It then asks for a pet or animal name.
3. Both values are combined into a band name and displayed on screen.

## Requirements

- Python 3.x (standard library only)
