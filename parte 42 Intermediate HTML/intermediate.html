How to Install Discord on Devices

The Basics

To make it work you need:

  • A laptop, PC, game console, or mobile phone
  • An internet connection
  • Access to a web browser
    • Note: if you do not have any of these, installation will not be possible
    • If your internet connection is unstable, the download may fail
    • If your browser does not open, you will not be able to download it

Instructions

  1. Go to the official Discord website
  2. Open the downloads and links section
  3. Open the downloaded file
  4. Once you have the file, install it by following the instructions shown
