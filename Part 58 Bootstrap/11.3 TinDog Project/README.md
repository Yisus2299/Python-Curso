# TinDog Project

Simple Bootstrap-based landing page template used as a learning exercise.

Contents
- Example HTML and CSS for a minimal landing page
- A few inline SVG icons used by the template

How to use
- Open `index.html` in a browser to preview the static template.
- Replace placeholder images and copy with your own content.

Notes
- This folder is an exercise in Bootstrap components and layouts; no server or backend is included.
