## HTML Concepts Covered

### 1. Basic Document Structure
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>
```

### 2. Heading Elements (h1-h6)
- `<h1>`: Main page heading (most important)
- `<h2>`-`<h6>`: Subheadings of decreasing importance
- Proper heading hierarchy for accessibility

### 3. Paragraphs and Text Formatting
- `<p>`: Paragraph elements
- `<br>`: Line breaks (empty element)
- `<hr>`: Horizontal rules (thematic breaks)
- Text organization and readability

### 4. Semantic HTML
- Proper use of heading hierarchy
- Logical document structure
- Accessibility considerations
- SEO-friendly markup

## Best Practices Demonstrated
1. **Document Type Declaration**: `<!DOCTYPE html>`
2. **Language Attribute**: `lang="en"` for English content
3. **Character Encoding**: `UTF-8` for international text
4. **Viewport Meta Tag**: Responsive design foundation
5. **Proper Nesting**: Correct element hierarchy
