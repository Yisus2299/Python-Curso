# Project 27 — Tkinter Practice

## Overview

Introductory Tkinter exercises and function parameter practice (`*args`, `**kwargs`).

## What Was Built

Three scripts: a basic Tkinter GUI with interactive widgets, and two function-focused exercises exploring default arguments, `*args`, and `**kwargs`.

## Concepts Covered

- Tkinter basics (window, labels, entries, buttons)
- `grid()` layout
- Button `command` callbacks
- Default parameters, `*args`, and `**kwargs`
- Class constructors with keyword arguments

## Files

| File | Description |
|------|-------------|
| `main.py` | Basic Tkinter GUI |
| `quizez.py` | Function argument exercises |
| `playground.py` | Advanced function and class exercises |

## How to Run

```bash
python main.py
python quizez.py
python playground.py
```

## How It Works

- **`main.py`:** Creates a window with a label, text entry, and button that updates the label when clicked.
- **`quizez.py`:** Practice with function signatures and argument passing.
- **`playground.py`:** Explores `**kwargs` in functions and class initialization.

## Requirements

- Python 3.x with Tkinter (standard library)
- A graphical environment for `main.py` (windowed display required)
