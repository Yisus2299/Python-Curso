# Project 5 — Password Generator

## Overview

A console program that generates random passwords based on user-defined character counts.

## What Was Built

A password generator that asks how many letters, numbers, and symbols to include, then builds a random password by sampling from each character set.

## Concepts Covered

- `for` loops
- `random.choice()`
- Lists of characters
- String concatenation

## Files

| File | Description |
|------|-------------|
| `project_5_caesar.py` | Main password generator |
| `part5_loops.py` | Practice exercises (loops, FizzBuzz, range) |

## How to Run

```bash
python project_5_caesar.py
```

## How It Works

1. The user specifies how many letters, numbers, and symbols they want.
2. The program randomly selects characters from each group.
3. All characters are combined and displayed as the final password.

## Requirements

- Python 3.x (standard library only)
