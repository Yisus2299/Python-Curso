# Project 4 — Rock Paper Scissors

## Overview

A console game where the player competes against the computer in Rock Paper Scissors.

## What Was Built

An interactive game that lets the user pick rock, paper, or scissors and compares the choice against a random computer selection to determine win, lose, or draw.

## Concepts Covered

- `random` module
- Lists and indexing
- Conditional logic for game rules

## Files

| File | Description |
|------|-------------|
| `project_4_rockPaperSicors.py` | Main game |
| `part4.py` | Practice exercises (random, nested lists) |

## How to Run

```bash
python project_4_rockPaperSicors.py
```

## How It Works

1. The user enters 0 (rock), 1 (paper), or 2 (scissors).
2. The computer picks a random option.
3. Standard Rock Paper Scissors rules determine the winner.

## Requirements

- Python 3.x (standard library only)
