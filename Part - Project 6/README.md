# Project 6 — Escape the Maze

## Overview

A simple console maze game where the player navigates from start to finish using directional commands.

## What Was Built

A mini-game that tracks the player's position on a linear path. The player moves left or right while avoiding walls until reaching the exit.

## Concepts Covered

- `while` loops
- State management with variables
- Conditional checks for boundaries and obstacles

## Files

| File | Description |
|------|-------------|
| `project_6_functions_while.py` | Main entry point |

## How to Run

```bash
python project_6_functions_while.py
```

## How It Works

1. The player starts at position 0 and must reach position 6.
2. They type "left" or "right" to move.
3. Positions 2 and 5 are walls — hitting them ends the game.
4. Reaching position 6 prints a victory message.

## Requirements

- Python 3.x (standard library only)
