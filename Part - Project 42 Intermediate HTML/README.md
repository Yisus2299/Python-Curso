## HTML Concepts Covered

### 1. Advanced List Structures
- Ordered lists (`<ol>`) with different numbering types
- Unordered lists (`<ul>`) with bullet styles
- Nested lists (lists within lists)
- Description lists (`<dl>`, `<dt>`, `<dd>`)

### 2. HTML Attributes
- Global attributes (`class`, `id`, `style`, `title`)
- Link attributes (`href`, `target`, `rel`)
- Image attributes (`src`, `alt`, `width`, `height`)
- Form attributes (`name`, `value`, `placeholder`)

### 3. Semantic HTML Elements
- `<header>`: Introductory content
- `<nav>`: Navigation links
- `<main>`: Main content area
- `<article>`: Independent content
- `<section>`: Thematic grouping
- `<aside>`: Sidebar/content
- `<footer>`: Footer information

### 4. Practical Webpage Structure
- Multi-section layouts
- Content organization
- Navigation structures
- Footer information

```html
<header>
    <h1>Website Title</h1>

            <li><a href="#contact">Contact</a></li>

    </nav>

<main>
    <article>
        <h2>Article Title</h2>
        <p>Article content goes here...</p>
    </article>
</main>
```
