# Project 3 — Treasure Island

## Overview

A text-based adventure game where the player makes choices that lead to different endings.

## What Was Built

A branching narrative game inspired by Treasure Island. Each decision (left/right paths, swim or wait, door colors) leads to a unique outcome — victory or game over.

## Concepts Covered

- `if` / `elif` / `else` conditionals
- Nested conditionals
- String comparison for user choices

## Files

| File | Description |
|------|-------------|
| `project_3_island.py` | Main entry point |

## How to Run

```bash
python project_3_island.py
```

## How It Works

1. The player arrives at a crossroads and chooses left or right.
2. Further choices determine whether they reach the treasure or fail.
3. Wrong decisions print a "Game Over" message; the correct path leads to victory.

## Requirements

- Python 3.x (standard library only)
