# Project 2 — Tip Calculator

## Overview

A console calculator that splits a restaurant bill and computes the tip per person.

## What Was Built

An interactive tip calculator that accepts the total bill, tip percentage, and number of people, then outputs the final amount and each person's share.

## Concepts Covered

- Type conversion (`float`, `int`)
- Arithmetic operations and division
- Formatted output with f-strings

## Files

| File | Description |
|------|-------------|
| `project_2.py` | Main entry point |

## How to Run

```bash
python project_2.py
```

## How It Works

1. Enter the total bill amount.
2. Choose a tip percentage (10%, 12%, or 15%).
3. Enter how many people are splitting the bill.
4. The program calculates the total with tip and the amount per person.

## Requirements

- Python 3.x (standard library only)
