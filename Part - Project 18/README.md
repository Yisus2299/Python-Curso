# Project 18 — Turtle Art (Hirst-Style Painting)

## Overview

Generative art projects using Python's Turtle graphics module, including a Damien Hirst-inspired dot painting.

## What Was Built

Two Turtle-based programs: a colorful dot grid inspired by spot paintings, and a practice file covering shapes, random walks, and spirograph patterns.

## Concepts Covered

- Turtle graphics (`Screen`, `Turtle`)
- `colormode(255)` for RGB colors
- Functions and loops for drawing patterns
- Random color selection

## Files

| File | Description |
|------|-------------|
| `proyecto18_GenerarPinturaRandom.py` | Hirst-style dot painting |
| `part18_POO_generateThings.py` | Turtle exercises and advanced patterns |

## How to Run

```bash
python proyecto18_GenerarPinturaRandom.py
```

## How It Works

1. A Turtle screen is set up with a grid layout.
2. Dots are placed in rows and columns across the canvas.
3. Each dot uses a randomly chosen color from a predefined RGB palette.
4. The practice file explores additional Turtle drawing techniques.

## Requirements

- Python 3.x with Turtle (standard library)
- Optional: `heroes` package for `part18_POO_generateThings.py` (`pip install heroes`)
- A graphical environment (windowed display required)
