# Project 27 — Miles to Kilometers Converter (Tkinter)

## Overview

A simple GUI application that converts miles to kilometers using Tkinter.

## What Was Built

A desktop converter with an input field for miles, a Calculate button, and a label that displays the equivalent distance in kilometers.

## Concepts Covered

- Tkinter widgets (`Entry`, `Label`, `Button`)
- `grid()` layout manager
- Button callbacks (`command`)
- Type conversion and arithmetic in event handlers

## Files

| File | Description |
|------|-------------|
| `main.py` | Main entry point |

## How to Run

```bash
python main.py
```

## How It Works

1. The user enters a distance in miles.
2. Clicking "Calculate" triggers the conversion function.
3. Miles are multiplied by 1.609 and rounded.
4. The result is displayed in the kilometers label.

## Requirements

- Python 3.x with Tkinter (standard library)
- A graphical environment (windowed display required)
